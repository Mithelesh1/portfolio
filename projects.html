Job Portal

The Job Portal is a web-based platform designed to bridge the gap between job seekers and recruiters. It allows employers to post job openings and candidates to apply directly by uploading resumes.

User Authentication: Separate login for Admin, Recruiters, and Applicants Job Listings: Employers can post, edit, and manage job openings

Resume Upload: Applicants can upload resumes and apply for jobs

Application Tracker: Employers can view and track job applications

Admin Panel: managing users, job categories, and overall portal control

Responsive UI: Mobile-friendly layout built using HTML/CSS

Tech Stack: HTML, CSS, PHP, MySQL

Pet Care Hub

Pet Care Hub is a web-based platform designed to connect pet owners with reliable pet care service providers such as walkers, sitters, and groomers. The platform streamlines the booking process and provides real-time availability for various pet services.

User-Friendly Interface: Clean and responsive design for easy navigation

Listings: Browse and filter pet care services by type and location

Real-Time Bookings: Schedule appointments and receive instant confirmation

Provider Profiles: View detailed profiles of pet walkers/sitters with ratings

Notifications: Automated service reminders and booking status updates

Tech Stack: HTML, CSS, JavaScript, PHP, MySQL

Image Editor App

The Image Editor App is a lightweight Android application designed for quick photo editing with powerful features tailored for everyday users. The app focuses on enhancing images through watermarking, background removal, and restoration — all while maintaining a simple and intuitive user interface.

Watermark: Insert custom text or logos as watermarks on photos

Background Removal: AI-assisted background erasing with preview feature

Photo Restoration: Enhance image quality, reduce blur, and adjust brightness

Offline Support: Most features work without internet connectivity

Tech Stack:PYTHON,HTML,CSS
